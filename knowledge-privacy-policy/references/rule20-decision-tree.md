# RULE20 知识处理路由决策树

## 最高原则

**RULE20 优先级高于所有其他 RULE 规则。** 每次响应前必须按此树审核。

## 决策流程

```
用户提问/任务请求
     │
     ▼
┌─────────────────────────────────────────────┐
│  问题涉及哪个知识域？                        │
└─────────────────────────────────────────────┘
     │
     ├── ZTLIG / Sinovatio(中新赛克) ──────→ ❌ 禁止在线 LLM
     │                                          │
     │                                          ▼
     │                                    记录到待办清单
     │                                    "等本地 LLM 建立后处理"
     │
     ├── A1 项目 ──────────────────────────→ ❌ 禁止在线 LLM
     │                                          │
     │                                          ▼
     │                                    全程本地处理
     │                                    read_file/search_files 仅限本地
     │
     ├── LI 协议/架构/概念 ───────────────→ ✅ 可用 OpenLI 替代后在线处理
     │                                          │
     │                                          ▼
     │                                    1. 确认有 OpenLI 对应概念
     │                                    2. 用 OpenLI 命名替代 LI 命名
     │                                    3. 可正常使用 DeepSeek
     │                                    例如: LIID → OpenLI 的 intercept ID
     │
     ├── HW(华为) 公开信息 ───────────────→ ✅ 可在线处理
     │     ZTE(中兴) 公开信息                   仅限公开文档/手册
     │     Ericsson(爱立信) 公开信息            不涉及具体项目配置/现场数据
     │     NSN(诺基亚西门子) 公开信息
     │
     ├── OWLS / SICMS / SECPASS ──────────→ ❌ 禁止在线 LLM
     │                                          │
     │                                          ▼
     │                                    等本地 LLM 建立后处理
     │
     └── 其他公开知识 ────────────────────→ ✅ 正常使用在线 LLM
```

## 豁免检查清单

在应用 RULE20 之前，逐项确认：

- [ ] 这个问题能直接用公开知识回答吗？（跳到"其他公开知识"）
- [ ] 涉及特定厂商的具体产品配置/现场数据吗？
- [ ] 涉及 A1 项目/苏丹/北苏丹的部署信息吗？
- [ ] 涉及 ZTLIG/Sinovatio 的二进制分析/架构/配置吗？
- [ ] 回答这个问题需要访问 knowledge/li/ 下的文件吗？

**任何一项为「是」→ 按对应规则处理，不默认走在线 LLM。**

## 与旧版 RULE6 的关系

RULE6 原文：禁止所有 LI/LIG/HW/ZTE/NSN/Ericsson/ZTLIG/Sinovatio/OWLS/SICMS/SECPASS 内容进入在线 LLM。

**RULE20 覆盖/细化了 RULE6：**
- ZTLIG/Sinovatio/OWLS/SICMS/SECPASS → 保留禁止（同 RULE6）
- HW/ZTE/Ericsson/NSN → 仅限公开信息可在线（放宽）
- LI 概念 → 用 OpenLI 替代后可在线（新增通路）
- A1 项目 → 全程禁止（同 RULE6 隐含）

**冲突时以 RULE20 为准。**
