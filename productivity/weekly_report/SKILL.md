# 周报生成 (weekly_report)

读取本周所有工作日志，按项目聚合，生成周报。

## 触发方式

- "生成本周周报"
- "周报按天分组"

## 默认格式

```markdown
## 工作周报 - {年份}年第{week}周（{start_date} - {end_date}）

### 各项目工时汇总
| 项目 | 总工时 | 主要工作 |
|------|--------|----------|
| {项目} | {工时} | {主要内容} |

### 本周重点工作
1. **{主要项目}**：{关键产出}

### 下周计划
- [ ] 
```

## 使用节奏

每周五下班前执行：`生成本周周报`

## WORD 文档生成路径

所有周报的 WORD 版本统一保存到：
```
~/sinovatio/周报/maohengzhen_周报_YYYY年第WW周.docx（文件名和文档标题均带 maohengzhen 前缀）
```

## 工作流

1. 读取本周（周一至周五）所有工作日志
2. 按项目聚合，计算各项目总工时
3. 提取各项目主要工作内容
4. 按工时排序输出
5. 可选包含下周计划
6. **生成 WORD 文档** 保存到 `~/sinovatio/周报/`
