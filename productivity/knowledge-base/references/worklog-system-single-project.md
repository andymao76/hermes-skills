## 使用方式

### 日常记录（不写项目名 → 自动归到默认项目）

| 场景 | 对 Hermes 说 | 示例 |
|------|-------------|------|
| 开始任务 | `开始任务：{工作内容}` | 开始任务：整理结项文档 |
| 结束任务 | `结束任务，用了X小时` | 结束任务，用了1.5小时 |
| 快速记录 | `记一下：{内容}，X小时` | 记一下：准备验收材料，2小时 |
| 补记 | `补记 {时间} {内容} {工时}` | 补记昨天下午 客户沟通 1小时 |

### 指定其他项目

```
记一下：埃及项目数据核对，1小时   → 埃及NSS-车载定位项目
记住，新项目'X'对应'Y'            → 自动添加映射
```

### 生成报表

| 命令 | 频率 | 写入路径 |
|------|:----:|---------|
| `生成今天的日报` | 每天下班前 | `工作/日报/YYYYMMDD.md` |
| `生成本周周报` | 每周五 | `工作/周报/` |
| `生成本月月报` | 每月最后一天 | `工作/月报/` |

## 路径说明（三分类结构）

- **Obsidian 原生路径**: `~/Documents/Obsidian Vault/工作/日报/YYYYMMDD.md`
- **kb-search 索引路径**: `~/知识/工作/日报/YYYYMMDD.md`（symlink `知识/ → ~/knowledge/` 下自动可见）
- **日期格式**: YYYYMMDD（如 20260611）
- **每日 06:00 cron 保活**: `bash ~/.hermes/scripts/ensure-vault-structure.sh`

## 多项目切换

当有新项目加入时，更新映射表和 project_status.yaml：

```
记住，新项目'埃及项目'对应'埃及NSS-车载定位项目'
```

记录规则：
- 不写项目名 → 默认项目
- 写了项目名 → 走映射表
- 匹配不到 → 询问用户
