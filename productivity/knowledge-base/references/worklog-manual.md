# 工作日志手册 (单项目模式)

## 核心流程（每天3步）

1. **上班** → `开始任务：{工作内容}`
2. **工作中** → 每次切换任务：`结束任务，用了X小时` + `开始任务：{新内容}`
3. **下班前** → `生成今天的日报`

## 常用命令

| 场景 | 命令 |
|------|------|
| 开始任务 | `开始任务：{内容}` |
| 结束任务 | `结束任务，用了X小时` |
| 快速记录 | `记一下：{内容}，X小时` |
| 补记 | `补记 {时间} {内容} {工时}` |
| 生成日报 | `生成今天的日报` |
| 生成周报 | `生成本周周报` |
| 生成月报 | `生成本月月报` |
| 调整格式 | `日报用列表格式` / `日报加上明日计划` / `按工时排序` |

## 项目映射（默认项目：A1 PC项目/苏丹NISS）

| 你说 | Hermes 理解成 |
|------|-------------|
| 印尼项目 | 印尼ASIOC项目 |
| 埃及项目 | 埃及NSS-车载定位项目 |
| 老挝项目 | 老挝MOPS-IFS升级项目 |
| 菲律宾项目 | 菲律宾AFFRS100A项目 |
| A1/苏丹/PC项目 | A1 PC项目（苏丹NISS） |
| 内部支撑 | 内部支撑工作 |

新项目用：`记住，"订单项目"就是PMO里的"销售订单处理"`

## 输出位置

`工作/日报/YYYYMMDD.md` → Obsidian vault 首层目录（三分类：工作/知识/技能）

## 相关技能

- `worklog` — 流水账记录
- `daily_report` — 日报生成
- `weekly_report` — 周报生成
- `monthly_report` — 月报生成
