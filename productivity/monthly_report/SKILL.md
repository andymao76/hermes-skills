# 月报生成 (monthly_report)

读取当月所有工作日志，生成月报。

## 触发方式

- "生成本月月报"
- "月报加上各项目的工时占比"

## 默认内容

1. 工时总览（当月累计工时）
2. 各项目工时明细
3. 项目状态变化
4. 关键产出总结
5. 下月工作重点

## 使用节奏

每月最后一天执行：`生成本月月报`

## WORD 文档生成路径

所有月报的 WORD 版本统一保存到：
```
~/sinovatio/月报/maohengzhen_月报_YYYY年MM月.docx（文件名和文档标题均带 maohengzhen 前缀）
```

## 工作流

1. 读取本月所有工作日志
2. 按项目聚合统计工时
3. 提取状态变化和关键产出
4. 按配置格式输出月报
5. **生成 WORD 文档** 保存到 `~/sinovatio/月报/`
