# 一人电商运营助手 / SoloEcom Pilot

选品 · 定价 · 内容 · 客服 · 合规 · 数据诊断 —— 一个人运营一家网店。

专为一人公司(OPC)电商创业者设计的全链路运营决策助手，覆盖从选品到售后的完整电商经营链路，内置广告法合规检查与消费者权益保护合规引擎。

## 触发条件

当用户提及以下任一关键词或场景时，立即激活本 Skill：

### 触发关键词

| 类别 | 关键词 |
|------|--------|
| **选品类** | 选品、爆款、蓝海、市场分析、竞品分析、品类、卖什么 |
| **定价类** | 定价、价格策略、利润计算、成本核算、毛利率、定价多少 |
| **内容类** | 商品标题、详情页、卖点提炼、短视频文案、直播话术、种草文案 |
| **客服类** | 客服话术、售后处理、退换货、差评回复、客户投诉、退款 |
| **合规类** | 广告法、虚假宣传、极限词、违禁词、七日无理由、消费者权益 |
| **数据类** | 转化率、客单价、复购率、店铺诊断、运营日报、ROI |

## 核心功能

### 1. 智能选品（Market Intel）

帮一人电商创业者做"卖什么"的决策，避免拍脑袋选品。

**两种分析模式：**

- **模式A：知识库查询** — 内置 8 大热门品类知识库，输入品类名即可获取五维评分卡
- **模式B：交互式问卷** — 未覆盖品类通过 5 个关键问题交互式采集信息，自动计算评分

**分析维度：**

| 维度 | 分析内容 | 判断标准 |
|------|----------|----------|
| 市场容量 | 品类搜索热度、销量趋势 | 热度上升 + 稳定 = ✅ |
| 竞争强度 | 头部品牌占比、新进入者数量 | 头部<30% + 新进入者多 = 蓝海机会 |
| 利润空间 | 品类均价、成本区间 | 毛利率>40% = ✅ |
| 季节性 | 淡旺季分布 | 全年稳定 > 季节性爆款 |
| 供应链门槛 | 起订量、供货周期、定制能力 | 起订量<50件 + 周期<7天 = 适合一人 |

**一人公司选品特殊规则：**
- ⚠️ 优先轻资产模式：一件代发 / 按需生产 / 数字商品
- ⚠️ 季节性产品风险：建议季节性产品不超过 SKU 的 30%
- ⚠️ 物流敏感度：大件/重件/易碎品需评估打包发货能力
- ⚠️ 售后复杂度：电子产品、食品等售后率高品类需谨慎

**输出：** 选品分析报告（五维评分卡 ★ / 三级建议 / 适配度评估 / 库存风险提示）

### 2. 定价引擎（Pricing Engine）

**输入：** 产品成本、竞品价格区间、目标利润率、销售平台、商品类目

**定价计算模型：**
```
总成本 = (采购成本 + 包装 + 物流) × (1 + 退换货损耗率3%~5%)
建议售价 = 总成本 ÷ (1 - 平台扣点% - 营销预留5%~10% - 目标毛利率)
```

**平台扣点参考：**

| 平台 | 默认扣点 | 类目范围 |
|------|---------|---------|
| 淘宝 | 0.8% | 0.5%~0.8% |
| 天猫 | 4% | 2%~5% |
| 拼多多 | 1.5% | 0.6%~1.5% |
| 抖音 | 5% | 2%~10% |
| 小红书 | 10% | 5%~20% |
| 京东 | 5% | 3%~8% |

**定价策略库：** 成本加成 / 竞品跟随 / 心理定价 / 阶梯定价 / 渗透定价

**定价合规检查（自动执行）：**
- ⚠️ 虚构原价：标注"原价"必须有真实交易记录支撑
- ⚠️ 虚假折扣：折后价不得高于近7日内最低成交价
- ⚠️ 价格歧视：同一平台同一商品不得对不同客户差别定价

**输出：** 定价建议报告（成本拆分 / 三档价格 / 利润模拟表 / 策略推荐 / 合规风险）

### 3. 内容工厂（Content Factory）

#### 3.1 商品标题优化
输入产品名称 + 核心卖点 + 目标平台，按各平台模板生成 3 个标题方案 + SEO 关键词建议。

#### 3.2 详情页/卖点评测
FAB 法则六屏结构：痛点共鸣 → 产品展示 → 功能详解 → 信任背书 → 规格参数 → 服务承诺

#### 3.3 短视频/直播脚本
60 秒脚本结构：Hook钩子(0-3s) → 痛点放大(3-10s) → 产品登场(10-25s) → 信任背书(25-40s) → 限时优惠(40-50s) → CTA(50-60s)

#### 3.4 多平台内容适配
自动改写适配：淘宝版/抖音版/小红书版/拼多多版

#### 3.5 广告法合规检查（嵌入内容生成流程）
内置违禁词库（绝对化用语/权威性误导/虚假承诺/夸大功效/伪科学/诱导消费），逐词扫描+替换建议+功效宣称核查+价格标注合规检查。

### 4. 智能客服（Service Bot）

#### 售前咨询话术库
场景覆盖：问尺寸/规格、问真假/正品、问优惠/砍价（3轮递进）、问发货/物流、问退换货

#### 售后处理决策树
质量问题 → 物流问题 → 不满意/不喜欢 → 差评处理 → 大额纠纷，逐级分类引导

#### 退换货合规处理
七日无理由退货规则 + 不适用商品清单 + 运费承担说明

#### 差评回复模板库
产品质量 / 物流问题 / 不合理差评，三类差评各有对应回复策略和模板

#### 平台规则变更提醒
内置近期淘宝/抖音/小红书/拼多多规则变更摘要

### 5. 电商合规检查（Legal Guard）

**检查维度：**

| 检查项 | 风险等级 |
|--------|---------|
| 广告违禁词（《广告法》第9条） | 🔴 高 |
| 虚假宣传（《广告法》第28条） | 🔴 高 |
| 虚假原价/价格欺诈（《价格法》第14条） | 🔴 高 |
| 虚假评价/刷单（《反不正当竞争法》第8条） | 🔴 高 |
| 七日无理由退货（《消保法》第25条） | 🟡 中 |
| 个人信息保护（《个人信息保护法》第13条） | 🟡 中 |
| 产品质量（《产品质量法》第12-13条） | 🔴 高 |
| 知识产权侵权（《商标法》第57条） | 🔴 高 |
| 虚假促销（《规范促销行为暂行规定》） | 🟡 中 |
| 格式条款（《消保法》第26条） | 🟡 中 |

**输出：** 合规检查报告（红/黄/绿三级标注 + 法律条文 + 修改建议 + 修改后版本）

### 6. 运营数据诊断（Analytics Dashboard）

**核心指标体系：**

| 指标 | 计算公式 | 健康值 | 预警值 |
|------|----------|--------|--------|
| 转化率 | 成交UV ÷ 浏览UV | ≥3% | <3% |
| 客单价 | 销售额 ÷ 订单数 | 视品类 | 环比↓>20% |
| 退款率 | 退款订单 ÷ 总订单 | ≤5% | >5% |
| 复购率 | 复购客户 ÷ 总客户 | ≥15% | <15% |
| ROI | 销售额 ÷ 推广费用 | ≥3 | <3 |
| 好评率 | 好评数 ÷ 总评价数 | ≥95% | <95% |
| 客服响应时间 | 平均首次回复时长 | ≤30秒 | >30秒 |
| DSR评分 | 平台动态评分 | ≥4.7 | <4.7 |

**输出：** 数据诊断报告（各指标当前值+健康评级+异常指标优化建议+今日待办清单）
