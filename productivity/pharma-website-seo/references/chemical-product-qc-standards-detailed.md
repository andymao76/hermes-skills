# 化学药品标签与质控标准参考（详细版）

> 综合整理自 ICH、FDA、NMPA、WHO 等权威指南。
> 用途：B2B API/中间体产品页「质量标准」板块的内容编写参考，以及客户信任度建设素材。

---

## 一、CAS 号（Chemical Abstracts Service）

**定义**：美国化学文摘社为每种化学物质分配的唯一数字标识符，避免命名混淆。

**编号结构**：
```
格式: PPPPP.PP.X
- 第一部分: 2~6 位数字
- 第二部分: 2 位数字
- 第三部分: 1 位校验码
示例: CAS 23579-79-5
```
数字越多通常表示结构越复杂。

**产品页用途**：
- URL 路径包含 CAS 号（SEO 核心策略）
- 产品表格首列突出 CAS 号
- 使用 Product + ChemicalSubstance Schema 标记

---

## 二、批号（Batch No.）

**通用约定**：批号 = 生产完毕日 YYYYMMDD
```
示例：生产 2023.08.21 → 批号 20230821
```
以**生产完毕日**为准（全部工序完成），非开始日。

## 三、有效期 vs 复验期（核心区别）

| 概念 | 英文 | 定义 | 法律效力 |
|------|------|------|---------|
| **有效期** | Expiry Date | 质量符合标准的最终期限 | **硬截止** — 超过后即使检验合格也不能使用 |
| **复验期** | Re-test Date | 需重新检验的日期 | **弹性间隔** — 复验合格后可继续使用并顺延 |

### 法规依据

- **ICH Q7A 推荐**："原料药通常用复验期，而不用有效期。"
- **中国 GMP (2010) 第165条**：物料质量标准须包含"有效期或复验期"
- **中国境内原料药**：按药品管理，只能标有效期；出口可依 ICH/FDA 标复验期

### 有效期计算方法

| 形态 | 有效期 | 示例（生产日 2020.01.10） |
|------|-------|------------------------|
| 固体/粉末 | +2年-1天 | 到期 2022.01.09 |
| 液体/油状 | +1年-1天 | 到期 2021.01.09 |
| 结晶类 | 根据熔点评估 | 熔点高→稳定→有效期长 |

### 进厂物料操作规则

| 标识情况 | 操作 |
|---------|------|
| 只有有效期 | 直接使用至有效期，一般无需复验 |
| 只有复验期 | 到期必须复验合格才能继续使用 |
| 两者同时 | 超过复验期但在有效期内，需复验 |
| 无明确有效期 | 企业自主制定复验期 |

> 复验期起算点为**进厂检验日期**，非生产/采购日期。

## 四、纯度（Purity）与含量（Assay）

- **两者一致**：都是混合物中目标组分的百分数
- 纯度越高，含量越高
- 产品页建议同时列出 purity 和 assay，并附 COA/HPLC 图谱

## 五、GMP 核心要求（ICH Q7 Framework）

| 指南 | 适用范围 | 核心内容 |
|------|---------|---------|
| ICH Q7 | API GMP | 质量体系、人员、厂房、设备、文件、物料管理、生产控制、实验室控制 |
| ICH Q11 | 原料药开发 | 工艺理解、杂质控制策略、过程重现性 |
| ICH Q3A/B | 杂质 | 有机/元素杂质限度 |
| ICH Q3C | 残留溶剂 | 溶剂限量 |
| ICH Q6A | 质量标准 | 检验项目和可接受标准 |
| ICH Q2 | 方法验证 | 准确度、精密度、专属性、线性 |

### API 质量标准应包含的项目
1. 外观（Appearance）
2. 鉴别（Identity）— NMR, IR, MS
3. 纯度（Purity）— HPLC/GC
4. 含量（Assay）
5. 残留溶剂
6. 重金属/元素杂质
7. 熔点/水分
8. 粒度分布（必要时）
9. 晶型（必要时）

### 供应商应提供的文档
- **COA** — 完整分析结果
- 方法验证报告
- 杂质路径分析
- 变更控制记录
- 稳定性数据
- DMF/CEP（法规申报用）
