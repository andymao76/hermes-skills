# 工作日志完整参考

## 命令速查

### 日常记录
| 场景 | 命令 |
|------|------|
| 开始任务 | `开始任务：{项目}，{内容}` |
| 结束任务 | `结束任务，用了X小时` |
| 快速记录 | `记一下：{项目}，{内容}，X小时` |
| 补记 | `补记 {时间} {项目} {内容} {工时}` |
| 查看今日 | `今天记了什么` |

### 生成报表
| 报表 | 命令 |
|------|------|
| 日报 | `生成今天的日报` |
| 周报 | `生成本周周报` |
| 月报 | `生成本月月报` |

### 格式调整
| 调整 | 命令 |
|------|------|
| 列表格式 | `日报用列表格式，不要表格` |
| 加明日计划 | `日报加上明日计划` |
| 按工时排序 | `按工时从多到少排序` |
| 周报按天 | `周报按天分组` |

## 单项目模式

默认项目：A1 PC项目（苏丹NISS）
不写项目名时自动归到此项目。写了项目名则走映射表匹配。

## 项目映射表
- A1项目/苏丹项目/PC项目 → A1 PC项目（苏丹NISS）
- 印尼项目 → 印尼ASIOC项目
- 埃及项目 → 埃及NSS-车载定位项目
- 老挝项目 → 老挝MOPS-IFS升级项目
- 菲律宾项目 → 菲律宾AFFRS100A项目
- 内部支撑 → 内部支撑工作

```

## 存储路径

```

## 典型工作日流程

```
09:30  开始任务：印尼项目，处理需求变更
11:30  结束任务，用了2小时
14:00  开始任务：埃及项目，数据核对
15:30  结束任务，用了1.5小时
16:00  记一下：帮汪岩处理订单问题，0.5小时
18:00  生成今天的日报
```

## 常见问题

| 问题 | 解决 |
|------|------|
| 忘记记工时 | `补记今天上午 印尼项目 写方案 2小时` |
| 记录错了 | `删除刚才那条记录` |
| 想看总工时 | `本周总工时多少` |
| 想看项目工时 | `印尼项目本月工时` |
