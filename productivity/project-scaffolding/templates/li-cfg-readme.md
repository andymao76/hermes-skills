## 关键参数

### NE 参数

| 参数 | 说明 |
|------|------|
| x1_ip/x1_port | X1 设控地址 |
| x1_user/x1_pwd | 网元认证凭据 |
| x2_ip/x2_transtype | X2 接口地址/传输方式 |
| x3_ip/x3_transtype | X3 接口地址/传输方式 |
| hw.alias | 网元别名 |
| hw.neid | 华为网元号 |

### VNE 参数

| 参数 | 说明 |
|------|------|
| vne_type | MSCs/IMS/SBC |
| operid | X2 报告运营商 ID |
| speechtype | 0=收发合并 1=分开发送 |
| incptType | 3=IRI+CC 1=仅 IRI |
