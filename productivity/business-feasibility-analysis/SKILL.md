# 创业/开店可行性分析报告生成器

对用户提供的创业项目或店面类型，进行系统化的多维度可行性分析，生成结构化深色主题 HTML 报告。

## 适用场景

- 用户提供店面/菜单图片 → 通过 `vision_analyze` 识别品牌和业态
- 用户描述想开什么类型的店 → 直接进入调研环节
- 需要综合评估：加盟/自创、租金、人员、税收、消防/安监、行业监管、财务预估

## 工作流

### 阶段一：业务识别

如果用户提供了图片（菜单、店面、品牌标识等）：

```
# 用 vision_analyze 识别图片内容
vision_analyze(image_url="图片路径", question="识别这是什么类型的店面/品牌，核心产品有哪些，价格区间，是否连锁品牌")
```

如果用户直接描述了开店意向，则跳过此步骤。

### 阶段二：多维度并行调研

使用 `web_search_plus` 并行搜索以下维度（每次调用单独搜索，不要合并关键词）：

| 维度 | 搜索关键词模式 | 说明 |
|------|---------------|------|
| **品牌加盟** | `{品牌名} 加盟 费用 条件 2026` | 加盟费、总投资、品牌背景 |
| **商铺租金** | `{城市} 餐饮店 租金 商铺 价格 2026` | 分商圈/社区底商/购物中心 |
| **地方税收** | `{城市} 个体工商户 税收 政策 2026` | 增值税减免、个税减半、核定征收 |
| **用工成本** | `{城市} 餐饮 人员工资 用工成本 2026` | 最低工资标准、各岗位薪资范围 |
| **消防审批** | `{城市} 餐饮店 消防要求 审批 2026` | 面积门槛、告知承诺制、检查要求 |
| **行业监管** | `{城市} 餐饮 营业执照 食品经营许可证 办理` | 证照流程、健康证要求、合规要点 |
| **地方政策** | `{城市} 餐饮 扶持政策 2026` | 地方政府对餐饮业的扶持措施 |
| **行业趋势** | `餐饮行业 现状 数据 2026` | 行业报告、开关店率、连锁化率 |

### 阶段三：数据综合与财务建模

收集完数据后计算：

- **启动资金汇总**：加盟费 + 装修 + 设备 + 首批进货 + 房租(押一付三) + 首月人工 + 证照费用 + 流动资金
- **月度经营预估**：日均营业额 × 30天 = 月营收，再逐项扣减原材料(38-42%)、人工(28-32%)、房租(5-10%)、水电(4-6%)、税费(1-3%)、其他(2-4%)
- **回本周期**：启动资金 ÷ 月净利润

### 阶段四：生成 HTML 报告

**模板结构：**

```html
<!DOCTYPE html>
<html lang="zh-CN">
<head>
  <!-- Dark theme (#0a0e17 background), 中文友好字体 -->
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>{城市}{业态}开店可行性分析报告</title>
  <style>
    /* 深色主题配色方案 */
    body { background: #0a0e17; color: #e0e0e0; }
    .header h1 { color: #f0c040; }  /* 金色标题 */
    .section { background: #111a26; border: 1px solid #1e2d3d; }
    .section h2 { color: #f0c040; border-bottom: 2px solid #1e2d3d; }
    .card { background: #0f1a28; border: 1px solid #1a2d40; }
    .big-number { color: #4fc3f7; }  /* 浅蓝数字 */
    .tag-green { background: #1b5e20; color: #a5d6a7; }
    .tag-orange { background: #e65100; color: #ffcc80; }
    .tag-blue { background: #0d47a1; color: #90caf9; }
    th { background: #0d1520; color: #f0c040; }
    tr:hover td { background: #0f1a28; }
    .pro-list { background: #0d2a1a; border: 1px solid #1b5e20; }
    .con-list { background: #2a1a1a; border: 1px solid #5e2020; }
  </style>
</head>
<body>
  ...
</body>

```

报告包含以下模块（按顺序）：

1. **头部**：标题、副标题、生成日期
2. **业务识别**：图片分析结果（如通过图片识别）
3. **品牌加盟分析**：品牌背景、加盟费用表（按城市等级）
4. **启动资金汇总**：用卡片式布局展示各项费用，底部显示总金额
5. **商铺租金**：社区底商 vs 商圈租金对比
6. **人员配置**：岗位×人数×月薪表，分岗位明细 + 合计行
7. **税收政策**：增值税优惠、个税优惠、地方扶持、六税两费
8. **消防与监管**：消防分类管理、证照流程表（步骤×事项×时间×费用）
9. **月度经营预估**：营收→成本→净利润逐项明细表，净利率标注
10. **优劣势分析**：pro-con 双栏布局
11. **决策建议**：编号列表，实操性建议

## 关键数据源优先级

| 数据类型 | 推荐来源 |
|----------|---------|
| 加盟信息 | canyinqx.com, 91jm.com, 品牌官网 |
| 商铺租金 | wx.51mingshun.com, 安居客, 58同城 |
| 税收政策 | 地方政府网站 (wuxi.gov.cn), 税务总局公告 |
| 薪资数据 | 职友集 (jobui.com), 人社部最低工资标准 |
| 行业报告 | 毕马威餐饮报告, 美团研究院, 中国饭店协会 |
| 消防规定 | 国家消防救援局 (119.gov.cn), 省住建厅文件 |
| 食安法规 | 市场监管总局, 食品经营许可管理办法 |

## 注意事项与陷阱

1. **不要一次搜索所有维度** — 每个维度单独搜索，结果更精准
2. **加盟数据多为第三方平台** — 加盟费用需联系品牌方核实，平台数据可能夸张
3. **当前餐饮市场风险较高** — 务必在报告中提示：2024年新店一年内关门率约46%
4. **个体工商户税收优惠至2027年底** — 2026-2027年是个体户开店税收窗口期
5. **消防分类管理按300㎡分界** — ≤300㎡适用告知承诺制，显著简化流程
6. **食品经营许可证有效期5年** — 需提前20个工作日申请，流程约10-20天
7. **健康证是所有员工必需** — 无健康证上岗罚款 ¥1,000-10,000（典型案例无锡2026）
8. **江苏2026新政大力简化餐饮审批** — 装修消防集成办理、食品许可一网通办、首违不罚
9. **月度预估表中的原材料占比要按业态调整** — 白斩鸡/鸡类约38-42%，火锅约35-40%，奶茶约25-30%
10. **流动资金的估算要保守** — 至少预留6个月运营资金，按"月总支出×6"计算
11. **加盟 vs 自创品牌的选择** — 加盟费可比作"买流量"，自创品牌节省加盟费但需自己打造品牌认知
