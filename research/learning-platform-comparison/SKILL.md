# 在线学习/技能提升平台对比

## 按需求推荐

| 目标 | 最佳平台 | 费用 | 评分 |
|:-----|:---------|:----:|:----:|
| 🧮 K-12数学/科学基础 | **Khan Academy** | 完全免费 | ⭐⭐⭐⭐⭐ |
| 🔬 AP/SAT 备考 | **Khan Academy**（官方合作） | 完全免费 | ⭐⭐⭐⭐⭐ |
| 💻 编程/Web开发 | Codecademy / The Odin Project | 部分免费 | ⭐⭐⭐⭐ |
| 🏢 职业/商务/管理 | Coursera / LinkedIn Learning | 部分免费 | ⭐⭐⭐⭐ |
| 🎓 大学深度课程 | MIT OCW / edX | 免费听课 | ⭐⭐⭐⭐⭐ |
| 🤖 AI/机器学习 | fast.ai / DeepLearning.AI | 免费 | ⭐⭐⭐⭐⭐ |
| 🗣 语言 | Duolingo | 部分免费 | ⭐⭐⭐⭐ |
| 🧪 互动式数学/科学 | Brilliant | 付费 | ⭐⭐⭐⭐ |
| 🎬 创意/设计 | Skillshare / Domestika | 付费 | ⭐⭐⭐⭐ |

## 推荐组合策略

> 打基础 → **Khan Academy**（免费，数学/科学/AP）
> 学编程 → **Codecademy + The Odin Project**
> 学 AI → **fast.ai + DeepLearning.AI**（免费）
> 拿学位 → **edX / Coursera**（付费证书）
> 学语言 → **Duolingo**（免费基础 + 付费高级）

## Khan Academy 优势

- 全球最大的免费K-12教育资源库，75,000+ 练习题
- Khanmigo AI 私教（2025+，GPT-4驱动，持续迭代）
- SAT/AP官方备考合作，完全免费
- 无广告、无订阅、无隐藏收费

## Khan Academy 局限

- 职业技能/商务管理内容几乎为零
- 编程深度不如 Codecademy 等专业平台
- 大学高阶内容不如 MIT OCW / edX
- 非英文内容极少
