# 技能提升 / 学习平台对比指南

## 触发条件

- 用户问 "可汗学院好吗"、"最好的学习平台"、"怎么学XX技能"、"Khan Academy" 等
- 需要对比不同在线学习平台
- 想根据学习目标选择最佳平台

## 核心平台对比

| 你想学什么 | 最佳平台 | 费用 | 特点 | 评分 |
|:-----------|:---------|:----|:-----|:----:|
| 🧮 K-12数学/科学 | **Khan Academy** | ✅ 完全免费 | 系统化课程、练习+视频 | ⭐⭐⭐⭐⭐ |
| 📝 SAT/AP 备考 | **Khan Academy** | ✅ 完全免费 | 官方合作，College Board认可 | ⭐⭐⭐⭐⭐ |
| 💻 编程入门 | **Codecademy** / **The Odin Project** | 部分免费 | 交互式编码，项目驱动 | ⭐⭐⭐⭐ |
| 🏢 职业/商务技能 | **Coursera** / **LinkedIn Learning** | 部分免费 | 大学认证，企业级 | ⭐⭐⭐⭐ |
| 🏛 大学深度课程 | **MIT OCW** / **edX** | 免费听课+付费证书 | 顶级大学课程 | ⭐⭐⭐⭐⭐ |
| 🤖 AI/机器学习 | **fast.ai** / **DeepLearning.AI** | 免费 | 实践导向、顶级讲师 | ⭐⭐⭐⭐⭐ |
| 🗣 语言学习 | **Duolingo** | 部分免费 | 游戏化、碎片时间 | ⭐⭐⭐⭐ |
| 🎨 设计/创意 | **Skillshare** / **Canva Design** | 付费 | 项目制，社区反馈 | ⭐⭐⭐ |
| 📊 数据分析 | **DataCamp** / **Kaggle Learn** | 部分免费 | 实战+竞赛 | ⭐⭐⭐⭐ |

## 选型决策树

```
我的学习目标是？
├─ K-12 课业/考试 → Khan Academy
├─ 专业编程/开发
│  ├─ 零基础 → Codecademy / The Odin Project
│  └─ 已会基础 → 官方文档 + GitHub + LeetCode
├─ AI/数据科学
│  ├─ 零基础 → Kaggle Learn / fast.ai
│  └─ 进阶 → DeepLearning.AI / 论文直接啃
├─ 大学级别深度学习 → MIT OCW / edX
├─ 职业技能/证书
│  ├─ 想被认可 → Coursera（大学认证）
│  └─ 想快速上手 → LinkedIn Learning
├─ 语言 → Duolingo + 沉浸式（Italki/Netflix）
└─ 综合/不确定 → 从 Khan Academy 开始（免费且全面）

注意：没有"一个平台搞定所有"。组合使用才是最优解。
```

## 可汗学院深度解析

| 项目 | 说明 |
|:-----|:------|
| 内容范围 | K-12数学、科学(生物/化学/物理)、编程、历史、经济、SAT/AP |
| AI 能力 | **Khanmigo** — AI辅导老师，2025-2026持续升级 |
| 费用 | **完全免费**（全球独此一家） |
| 局限 | ❌ 无职业技能类课程 ❌ 无认证/证书 ❌ 无大学高级课程 |
| 最佳搭档 | Coursera（证书）+ Codecademy（编程）+ Khan Academy（基础） |

## 2026年学习策略建议

1. **基础打底** → Khan Academy（免费、系统、无广告）
2. **专项突破** → 目标导向选专业平台（AI→fast.ai, 编程→The Odin Project）
3. **实战落地** → GitHub/LeetCode/Kaggle 真实项目
4. **认证加持** → Coursera/edX 行业认证
