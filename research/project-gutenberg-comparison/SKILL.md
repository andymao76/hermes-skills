# 古腾堡计划(Project Gutenberg) 免费数字图书馆对比

## 触发条件
- 用户问"古腾堡计划好不好"、"最好的免费图书馆"、"免费电子书"

## 古腾堡计划评价

| 维度 | 评分 | 说明 |
|:-----|:----:|:-----|
| 历史地位 | ⭐⭐⭐⭐⭐ | 1971年创立，全球最老数字图书馆 |
| 藏书量 | ⭐⭐⭐⭐ | 75,000+本(2026) |
| 免费程度 | ✅ **完全免费** | 无广告/无付费墙/无需注册 |
| 格式支持 | ⭐⭐⭐⭐ | EPUB/Kindle/HTML/纯文本 |
| 可及性 | ⭐⭐⭐⭐⭐ | 无需账号，直接下载，离线阅读 |
| 现代性 | ⭐⭐ | 只限公有领域作品(美国1929年前) |
| 非英文内容 | ⭐⭐ | 以英文为主 |
| 搜索/发现 | ⭐⭐⭐ | 基本可搜 |

## 核心局限
- **只收录公有领域作品**，无现代畅销书
- 美国版权法限制1929年后作品不可收录
- 无社交/阅读追踪功能

## 替代平台对比

| 平台 | 藏书量 | 收费 | 特色 |
|:-----|:------:|:----:|:-----|
| **Project Gutenberg** | 75,000+ | 免费 | 经典文学，无需注册 |
| **Standard Ebooks** | 1,000+ | 免费 | 精校+精美排版 |
| **Internet Archive** | 20,000,000+ | 免费 | 全品类(含现代书借阅) |
| **Open Library** | 3,000,000+ | 免费(借阅制) | 现代书可借阅 |
| **LibriVox** | 15,000+ | 免费 | 公版书有声书 |
| **Z-Library** | 12,000,000+ | 部分免费 | ⚠ 灰色地带，覆盖最全 |
| **Anna's Archive** | 聚合搜索 | 免费 | 元搜索引擎 |

## 最佳组合
- 经典文学 → **Project Gutenberg** + **Standard Ebooks**
- 有声书 → **LibriVox**
- 绝版/学术书 → **Internet Archive**
- 现代书借阅 → **Open Library**
- 搜书元搜索 → **Anna's Archive**
