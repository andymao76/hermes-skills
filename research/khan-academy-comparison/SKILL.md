# 可汗学院(Khan Academy) 技能提升平台对比

## 触发条件
- 用户问"可汗学院好不好"、"最好的学习平台"、"技能提升方法"

## 可汗学院评分

| 领域 | 评分 | 说明 |
|:-----|:----:|:-----|
| K-12数学 | ⭐⭐⭐⭐⭐ | 免费、系统化、全球最强，从算术到微积分 |
| 科学(物理/化学/生物) | ⭐⭐⭐⭐ | AP级别，深度够 |
| SAT/AP备考 | ⭐⭐⭐⭐⭐ | 与College Board官方合作，免费 |
| 编程入门 | ⭐⭐⭐ | JS/HTML/Python/SQL基础 |
| AI辅导(Khanmigo) | ⭐⭐⭐⭐ | GPT-4驱动，解题引导不直接给答案 |
| 职业技能 | ⭐ | 几乎没有商务/管理类课程 |
| 大学高阶 | ⭐⭐ | 不如MIT OCW或Coursera专项课程 |

## 平台对比：学什么用哪个

| 目标 | 最佳平台 | 费用 |
|:-----|:---------|:-----|
| 数学/科学基础 | **Khan Academy** | ✅ 免费 |
| 编程/开发 | **Codecademy / The Odin Project** | 部分免费 |
| 职业/商务技能 | **Coursera / LinkedIn Learning** | 部分免费 |
| 大学深度课程 | **MIT OCW / edX** | 免费听课 |
| AI/机器学习 | **fast.ai / DeepLearning.AI** | 免费 |
| 语言学习 | **Duolingo** | 部分免费 |
| 互动数学思维 | **Brilliant** | 付费 |

## 注意事项
- 可汗学院**不是**职业技能提升平台
- Khanmigo AI 辅导($44/年或$15-20/月)质量优于大多数AI学习工具
- 中文内容有限，适合英文用户
