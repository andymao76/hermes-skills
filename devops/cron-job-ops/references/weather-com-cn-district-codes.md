# weather.com.cn 区县级天气代码参考

中国天气网 (weather.com.cn) 为城市及区县提供独立天气页面，通过 9 位数字代码访问。

## URL 格式

```
https://www.weather.com.cn/weather/<CODE>.shtml
```

返回内容包含 7 天天气预报表格、生活指数、24 小时实况等，适合 web_extract 解析。

## Cron Job 中使用

### 带防重复机制的天气日报（推荐模式）

对于每天定时发送的天气任务，使用 **retry window + 防重复标记** 模式：

**Schedule:** `*/30 7-9 * * *`（7:00~9:30每30分钟重试一次）
**Tools:** `["web", "terminal"]`（terminal 用于读写标记文件）

**Prompt 模板（紧凑格式，仅当天）：**

```markdown
你的任务是查询XX各区当天的天气。

## 防重复
每次运行前，用 terminal 检查：
cat /home/user/.hermes/.weather_sent_$(date +%Y%m%d) 2>/dev/null
文件存在则跳过。

## 查询
web_extract 批量查询以下URL，仅取当天（表格第一行）。

## 输出格式（紧凑，一行一个区）
========== 🌤 城市 今日天气 ==========
📅 6月10日 周三 | 中央气象台

── 城市名 ──
【市区】多云转阴 19~32°C
【某区】阴 20~31°C
...
── 提示 ──
最高温/降雨/大风/防晒

## 标记
echo "sent at $(date)" > /home/user/.hermes/.weather_sent_$(date +%Y%m%d)
```

### 用户偏好：紧凑输出

用户偏好**紧凑格式**（一行一个区，无表格线），而非完整表格。关键信息浓缩：天气、温度范围、风力（仅当≥3级时标注）。按区/市分组，以简短提示收尾。

### 多区查询

分批调用 web_extract（每次最多 5 个 URL 以避免超时），从返回内容中解析当天天气预报行。

## 数据提取要点

- **当天天气**在表格第一行，格式：日期 | 天气 | 温度 | 风力
- **温度格式**：`19~32°C` 或 `32°C / 19°C`（min/max）
- **风力字段**可能包含 `3-4级转<3级` 等复合值
- 更新时间为页面标题下的 "更新时间：YYYY-MM-DD HH:MM"
- 只需取第一行（当天），后面的行是未来几天的预报
