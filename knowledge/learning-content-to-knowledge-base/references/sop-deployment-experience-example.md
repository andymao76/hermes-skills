## 入库决策过程

1. **题材判断**: 运维/配置经验类文档 → 适合 `ops/` 目录
2. **重叠检查**: 已有 `ops/hermes-win11-remote-llm-sop.md`（Win11 远程 LLM 部署），内容互补
3. **目录确定**: 放入 `ops/` 同目录，补充而非替代
4. **交叉链接**: 互加 `关联文档` wikilink

## 关键策略

| 策略 | 说明 |
|------|------|
| 结构化优先 | 将自由文本转化为 SOP 格式（现象→根因→修复三列模型） |
| 交叉链接优先于重复 | 不要重复写入已有文件已有内容，用 wikilink 引用 |
| 双向链接 | 新文件引用旧文件 + 旧文件更新回链 |
| 表格化 | 对比表、环境表、验证清单表替代叙述段落 |
| 元数据可追溯 | 版本、来源、更新日期、关联文档都在顶栏 |
