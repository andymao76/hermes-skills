# 高质量输出示例集

> 本文件收录各模块的标准输出示例，供评测参考和用户预期对齐使用。

## 模块 01 · 内容创作

### 示例：小红书种草文

**用户输入：**
> 帮我写一篇关于"早起习惯"的小红书文章，目标读者是25-35岁职场女性，1000字左右

**标准输出：**
```
【适用平台：小红书 | 字数：约950字 | 语气：亲切分享型】

🌅 坚持早起3个月，我的生活发生了这些变化

作为一个曾经把"早起"当笑话的人，我想聊聊这三个月真实发生的事。

【第一个月：痛苦期，别信那些"越早越精神"的鬼话】
凌晨5点半的闹钟响起，我的第一反应永远是——凭什么？
没有任何灵感涌现，没有"一天最清醒的时光"...
```

## 模块 02 · 商业运营

**用户输入：**

**标准输出：**
```
📋 品牌定位画布 | 草木本

【品牌定位】天然植物手工皂的"成分党"首选品牌
【一句话描述】像读懂护肤品成分表一样，学会看懂皂

目标用户画像：
- 25-38岁女性，一二线城市
- 关注护肤成分，倾向天然有机产品
- 对工业香精敏感，追求简约生活方式
...
```

## 模块 03 · 产品研发

### 示例：PRD 文档

**用户输入：**
> 写一个用户登录功能的 PRD

**标准输出：**
```
# PRD: 用户登录功能

版本：v1.0
状态：草稿

## 背景与目标
支持邮箱/手机号+密码登录，以及第三方 OAuth 登录。

## 功能需求
1. 邮箱登录：【核心】输入邮箱+密码，验证后登录
2. 手机号登录：【核心】输入手机号+验证码，验证后登录
3. 第三方登录：【核心】支持微信/Google OAuth
...
```
