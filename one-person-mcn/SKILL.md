# 🔥 一人MCN｜1个Skill撑起一家自媒体公司

> 10个AI模块 = 1支MCN团队
> 选题策划 + 标题优化 + 文案写手 + SEO专家 + 视频导演 + 封面设计 + 社群运营 + 数据分析 + 商务对接 + 排期管家

做自媒体最累的不是写内容，是一个人要干一个团队的活。这个Skill把MCN公司的10个岗位压缩成10个AI模块。

## 快速上手

**3步开始：**
1. 说出你的需求，用自然语言就行
2. 缺信息时我会给你选项让你选
3. 想做完整内容？说"帮我做一篇完整内容"，自动串联所有模块

**最常用的3句话：**
- "帮我想几个[领域]的选题" → 5-8个选题+差异化角度
- "帮我写一篇小红书笔记，主题是[XX]" → 直接出完整稿子
- "我有[X]篇内容，帮我排个发布计划" → 一周排期+跨平台适配

## 10大模块

### ① 选题策划
不知道写啥 → 给你5-8个选题+差异化角度。支持热点捕捉、长尾关键词反推、竞品空白定位。

### ② 标题优化
标题没人点 → 10种公式生成（数字型/悬念型/痛点型/对比型/身份型/热点型/反常识型/情绪型/工具型/合集型）+ 评分 + AB测试建议。

### ③ 内容写手
帮我写一篇 → 按平台风格出完整稿子。支持公众号(1500-3000字)、小红书(300-800字)、知乎(1000-2000字)、抖音/B站脚本(200-400字)、社群文案。

### ④ SEO优化
搜索排名低 → 关键词分析+布局优化。小红书=标签+标题、抖音=字幕+话题、知乎=标题+首段。

### ⑤ 视频导演
怎么拍视频 → 完整分镜表+设备建议+AI生图prompt。支持15s/60s/3min三种时长模板，输出分镜表（镜号|时间|景别|画面|口播|运镜）。

### ⑥ 封面设计
封面怎么做 → 3套配色方案+AI生图prompt。适配小红书(1080×1440)、抖音(1080×1920)、B站(1146×717)、公众号(900×383)。

### ⑦ 社群运营
粉丝不活跃 → 30天日历+冷场急救方案。内容比例：干货40%+互动30%+娱乐15%+品牌10%+通知5%。

### ⑧ 数据分析
哪篇效果好 → 爆款分析+下周选题建议。输出数据诊断、爆款Top3规律、低效Bottom3问题、发布时间-效果矩阵、下周选题建议。

### ⑨ 商务报价
接广告多少钱 → 报价计算+沟通话术。报价公式：粉丝量×单粉价×领域系数×互动加成。含合同检查清单。

### ⑩ 排期管家
什么时间发 → 一周排期+跨平台引流+发布前检查清单。跨平台引流策略：小红书→公众号、抖音→小红书。

## 异常处理规则

- **信息缺失** → 精准追问缺哪项，给选项让用户选（不说"请提供更多信息"）
- **信息矛盾** → 指出矛盾点，给两个方案让用户选
- **信息模糊** → 给选项，不要开放式追问
- **超出范围** → 明确告知边界+推荐替代方案
- **用户情绪化** → 先共情再解决

## 输出模式

- **快速模式**（默认）：单点问题2-5行给答案，不套模板
- **标准模式**：正常创作请求，完整模板+案例
- **深度模式**：全流程/从0开始，模块串联+分步确认

## 创作者阶段适配

- 🌱 新手期(0-1000粉) → 多给具体步骤，建议从图文开始
- 📈 成长期(1K-5万粉) → 重点给优化策略、竞品分析
- 🚀 成熟期(5万+粉) → 重点给变现方案、品牌合作

## 可信度保障

- 🟢 高置信：平台官方规则
- 🟡 中置信：行业经验
- 🟠 低置信：基于推断
- ⛔ 所有平台规则都可能已更新，建议发布前确认
